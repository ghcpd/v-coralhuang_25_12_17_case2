Usage Guide

  • Search: type keywords into the search box; matches in titles are highlighted.
  • Filter: choose a status from the dropdown to filter results.
  • Sort: click column headers to sort; clicking cycles asc → desc → none. Arrow indicates direction.
  • Pagination: use Prev/Next and change page size.
  • Inline status updates: click Start or Complete on applicable rows to update status. A row-level spinner shows while updating; success briefly highlights the row; failures show an inline error message.
  • Reset: click Reset to clear search, filter, sort and pagination back to defaults.

Usage Guide

Search Tasks

Use the search box to find tasks by title. Matching text will be highlighted in the results.

Filter & Sort

Filter tasks by status using the dropdown. Click column headers to sort by that field.

Update Status

Click the status dropdown in any row to update task status. Status must progress: TODO → IN_PROGRESS → DONE.

Navigate Pages

Use the pagination controls at the bottom to browse through tasks. Adjust page size as needed.

View Details

Click any row to view full task details in a modal dialog.

Reset View

Use the Reset button to clear all filters, sorting, and return to the first page.
